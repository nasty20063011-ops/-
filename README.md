 Описание проекта
Проект представляет собой психолингвистическое исследование, посвященное тому, как человек различает философские высказывания и цитаты из песен.

 Основные метрики
Чувствительность (d') = 0.11 (низкая способность различать стимулы)
Критерий принятия решения  = -0.45 (либеральный, склонность отвечать "Да")
Вероятность ложной тревоги = 0.8 для цитат-дистракторов
